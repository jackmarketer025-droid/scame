# **App Name**: Scam Shield Command

## Core Features:

- Smart Verification Engine: Allows users to verify website URLs or mobile numbers by checking against internal blacklists, domain age via Whois API, and global databases like Google Safe Browsing/VirusTotal API.
- AI-Powered Verification Insight: Utilizes generative AI as a tool to provide user-friendly explanations and nuanced risk assessments for verification results, especially for 'Suspicious' entries, offering actionable advice.
- Multi-Step Scam Reporting: Guides users through a multi-step form to report scams, capturing scam type, scammer information, story details, and allowing for evidence (screenshots, audio, video) upload.
- Searchable Scam Database: Provides a filtered and searchable table of approved scam reports, allowing users to browse by date, scam type, platform, and status (Active/Closed).
- Interactive Scam Heatmap: Visualizes scam reporting data on a dynamic map of Bangladesh, showing district-wise scam statuses and trends.
- Awareness Resource Hub: Centralizes educational content including articles ('How to spot fake websites?'), integration of the latest YouTube videos, and downloadable awareness materials (posters, leaflets).
- Admin Moderation Panel: A dedicated dashboard for administrators to review and moderate incoming scam reports, approve them for public display, reject/delete fraudulent entries, and manage live alerts.

## Style Guidelines:

- Primary color: Deep Blue (#003366), evoking trust, security, and professionalism. This is used for navigation, core elements, and key branding.
- Background color: A very subtle, light blue (#F3F8FA), derived from the primary blue's hue, provides a clean, expansive canvas while maintaining a cohesive visual identity.
- Accent color: Warning Red (#E63946), dynamically employed for emergency alerts, 'Scam' status indicators, and primary call-to-action buttons like 'Emergency Alert', signaling urgency.
- Secondary color: Pure White (#FFFFFF), used for content cards, interactive elements, and text for optimal contrast and readability within the light theme.
- Headlines and prominent text will use 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, set at approximately 32px. Body text will utilize 'Poppins' (sans-serif), a precise and contemporary geometric sans-serif, at 16px for comfortable readability. Note: currently only Google Fonts are supported.
- Utilize clean, modern, and easily recognizable vector icons throughout the interface to clearly communicate actions, statuses, and content categories, supporting the portal's serious function.
- Implement a mobile-first responsive modern card design system to ensure an intuitive and accessible user experience across all devices, emphasizing clarity and efficient content presentation.
- Incorporate subtle and purposeful animations for user feedback, such as state changes, alert notifications, and interactive elements, to enhance engagement without detracting from critical information.
